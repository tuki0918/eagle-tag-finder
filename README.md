# Eagle Tag Finder

A plugin that helps you quickly search Eagle tags and browse them with thumbnails.

> [!WARNING]
> Thumbnails do not show up on Eagle v4.0.0 Build17 (2025-12-03).

## Features
- Search tags with suggestions
- Click a tag card to copy the tag name to the clipboard
- Filter tags (All / Starred / Recent)
- Filter tags by group

## Development Notes
The following features are not available due to API limitations:
- Add/remove tags to favorites (starred)
